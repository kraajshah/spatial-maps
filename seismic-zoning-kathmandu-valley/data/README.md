# Data notes
The source datasets used for this map are provided by the **Department of Mines and Geology (DMG), Kathmandu, Nepal** and are **not uploaded** here due to access and sharing restrictions.

## Coordinate reference system (CRS)
- **CRS:** WGS 84 (EPSG:4326)
- Units: degrees (latitude/longitude)

This repository includes **map outputs and workflow notes only** to demonstrate the GIS method and mapping approach.

If access is granted, the workflow can be reproduced using:
- Seismic zoning polygons
- Fault/thrust line features
- Administrative boundaries and reference layers
